| Problem | Other Tools | DotfileFlow |

### 🚀 Key Features

- **Declarative Configuration**: Define your ideal state in `dotflow.yaml`:
  ```yaml
  files:
    - src: ~/.dotfiles/zshrc
      dest: ~/.zshrc
      when: shell == zsh
    - src: ~/.dotfiles/nvim/init.lua
      dest: ~/.config/nvim/init.lua
      when: has(nvim)
  hooks:
    post-install:
      - nvim --headless +PlugInstall +qall
  ```

- **Profiles**: Switch between `work`, `personal`, `minimal`, or `gaming` setups with one flag.
- **Cross-Platform**: Native support for macOS, Linux, WSL2, and even Termux.
- **Backup & Rollback**: Every change is snapshotted. `dotflow undo` gets you back.
- **Diff-Aware**: See exactly what changed before applying — no surprises.
- **Community Powered**: Official registry of community-maintained configs (stars, themes, plugins, toolchains).
- **Zero Dependencies**: Single binary. Written in Rust. Blazing fast. No Python/Node/JVM needed.
